# Inferenza

An educational implementation of a minimal machine learning inference engine.

**Disclaimer:** This is a purely educational project. Its architecture, design philosophy, and core concepts are heavily inspired by [tinygrad](https://github.com/tinygrad/tinygrad). All credits for the clean and hackable architecture go to George Hotz and the tinygrad contributors.

## Goals
- Implement a minimal tensor library in Zig.
- Focus on high-performance CPU kernels using SIMD.
- Design a multi-backend architecture (CPU, CUDA, etc.).
